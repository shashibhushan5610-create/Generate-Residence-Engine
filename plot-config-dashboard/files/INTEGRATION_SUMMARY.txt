================================================================================
IF SOMETHING BREAKS
================================================================================

Error: "ModuleNotFoundError: No module named 'networkx'"
Fix: pip install -r requirements.txt

Error: "No module named 'layout_engine'"
Fix: Verify all 6 files are in backend/services/

Error: "POST /api/layouts/generate 404"
Fix: Verify you added blueprint registration in app.py (STEP 4)

Error: "Blank page in frontend"
Fix: Check browser console (F12) for JavaScript errors
     Usually missing import or file not in right location

Error: "POST request hangs / times out"
Fix: Backend might be solving a layout
     Large plots take 10-30 seconds
     Check Python console for progress logs
